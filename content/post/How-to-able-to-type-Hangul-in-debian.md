## How to 


Need to download `ibus-hangul`

1. typed below commands on Terminal
```
sudo apt-get update
sudo apt-get install ibus-hangul
```

2. After the installations, reboot.
`sudo reboot`

3. Go to `Setting`, `Region and Language`.
4. Add `Hangul`.
5. Tick the `start with hangul mode`
